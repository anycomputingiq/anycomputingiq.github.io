# 🚀 Deploy Your Website - Free Hosting Options

## Option 1: GitHub Pages (Recommended) - FREE
**Your website will be: `anycomputingiq.github.io`**

### Steps:
1. **Create GitHub Account** (if you don't have one)
   - Go to https://github.com/signup
   - Create account with username: `anycomputingiq`

2. **Create New Repository**
   - Go to https://github.com/new
   - Repository name: `anycomputingiq.github.io` (EXACT name)
   - Make it Public
   - Click "Create repository"

3. **Upload Your Files**
   - Click "uploading an existing file"
   - Drag and drop ALL files from `C:\Users\Public\aarij-portfolio\`
   - Files to upload:
     * index.html
     * styles.css
     * script.js
     * profile.jpg
     * teaching.jpeg
     * dev.jpeg
     * teach-2.jpeg
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to Settings > Pages
   - Source: Deploy from a branch
   - Branch: main
   - Click Save

5. **Done!** 
   - Your website will be live at: `https://anycomputingiq.github.io`
   - Wait 2-5 minutes for deployment

## Option 2: Netlify - FREE
**Your website will be: `anycomputingiq.netlify.app`**

### Steps:
1. **Sign Up**
   - Go to https://app.netlify.com/signup
   - Sign up with GitHub or Email

2. **Deploy Site**
   - Click "Add new site" > "Deploy manually"
   - Drag and drop the entire folder `C:\Users\Public\aarij-portfolio\`
   - Wait for deployment (30 seconds)

3. **Customize Domain**
   - Go to Site settings > Domain management
   - Click "Change site name"
   - Enter: `anycomputingiq`
   - Your site is now: `https://anycomputingiq.netlify.app`

---

## Option 3: Vercel - FREE
**Your website will be: `anycomputingiq.vercel.app`**

### Steps:
1. Go to https://vercel.com/signup
2. Sign up with GitHub
3. Click "Add New" > "Project"
4. Import from local files
5. Deploy!
